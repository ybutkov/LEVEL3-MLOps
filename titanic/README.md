# Titanic — Classification (week 1)

See [handout/README.md](handout/README.md) for the full assignment description.

## What's here

- `notebooks/01_titanic_eda.ipynb` — exploratory data analysis of the Titanic dataset
- `notebooks/02_scikit_learn_training.ipynb` — logistic regression with scikit-learn and model evaluation
- `notebooks/03_logreg_implementation.ipynb` — logistic regression from scratch using NumPy and gradient descent
- `src/logreg.py` — reusable implementation functions
- `tests/test_logreg.py` — unit tests for `src/logreg.py`
- `data/titanic.csv` — dataset
